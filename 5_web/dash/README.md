# Getting started with dash_html_components

<https://youtu.be/yfWJXkySfe0>

### Baseline

```
html.Div(
  [
  SOME_HTML_OR_DASH_COMPONENT(
    id,
    content(data),
    style(a dictionary with the properties)
    )
  ], className = (name of the Bootstrap Class)
  )
```

### We can include...**
- data
- layout
  - Title, color
  - position
